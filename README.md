## 🚀 Live Production System

### **[FreelanceFlow API Architecture](https://github.com/divyapratapdev/freelanceflow)** 
*Live API Documentation:* [**Swagger UI on Render**](https://freelanceflow-bxtt.onrender.com/swagger-ui/index.html)

My flagship backend engine built to mirror industrial SaaS architecture.
- **Tech Stack:** Spring Boot 3, Java 17, PostgreSQL, Redis, Apache Kafka, Spring Security 6 (JWT)
- **Key Engineering Decisions:**
  - Implemented **Distributed Idempotency** via Redis to prevent double-billing from Razorpay webhooks.
  - Event-Driven Architecture using **Kafka** to decouple Invoice processing and AI Analytics.
  - Hardened Deployment pipeline with environmental dynamic property resolution, passing successfully on Render Free-Tier constraints (512MB RAM).
